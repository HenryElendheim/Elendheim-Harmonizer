# Changelog

Everything worth noting about each version of Elendheim Harmonizer lives here.

**Version naming.** Releases are titled `Elendheim Harmonizer vX.Y`:

- Small steps bump the minor: v1.0 → v1.1 → v1.2 …
- A big milestone bumps the major: … → v2.0

Every release gets three plain lines describing what the app does, and — once
the build workflow has run — an APK attached to it.

## Elendheim Harmonizer v1.0 — 2026-07-13

The first release.

This app does three things:

- Listens to your voice through the mic.
- Stacks a perfect fifth on top in real time.
- Plays the blend back live as you sing.

Dark by default, one screen, MIT licensed.
